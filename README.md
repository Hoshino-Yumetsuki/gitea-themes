# 自用的好看的Gitea主题

## 安装

1. 将该css文件放在custom/public/assets/css下
Linux下custom目录在/var/lib/gitea下，若是docker请找到挂载的位置

2. 修改/etc/gitea/config/app.ini，找到[ui]，在THEMES字段添加新主题snowykami
3. 重启gitea服务，进入个人设置>外观>主题即可看到该主题
